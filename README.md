# nodejs-express-api-boilerplate

## Dependencies
- ckey 
- express
- body-parser
- mongoose
- concurrently (optional)

## DevDependencies
- nodemon

## Quick Start
- git clone https://github.com/shujo/nodejs-express-api-boilerplate.git
- create .env file for the mongodb config
- npm install
- npm run server or npm run start
